The official WordPress plugin, built by Teamacle.
Please note you will need PHP 5.6 or higher for this plugin.

Chat with visitors to your website in real-time, Teamacle provides everything your company needs for a Live Chat enabled website..

== Description ==

[Teamacle](http://www.teamacle.com/) is a fundamentally new way for internet businesses to communicate with customers, personally, at scale. It's a customer support platform, and Teamacle enables targeted communication with customers on your website, inside your web and mobile apps, and by email.

With this plugin, you can add the Teamacle Live Chat to your website in just a few clicks and start chatting to customers and visitors to your website right away.


== Installation ==

Installing Teamacle on your WordPress site takes just a few minutes.

For the installation instructions, click this link: [How to install on WordPress](http://teamacle.com/install-wordpress.html).

Note: This plugin injects a Javascript snippet on your website frontend containing dynamic user data. Some caching solutions will cache entire pages and should not be used with this plugin. Doing so may cause conversations to be delivered to the wrong user.
